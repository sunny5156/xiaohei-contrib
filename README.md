# xiaohei-contrib

[小黑](https://github.com/sunny5156/xiaohei) 



## 安装

``` sh
cd ~/.xiaohei
git clone https://github.com/sunny5156/xiaohei-contrib.git  contrib
pip install -r contrib/requirements.txt
```

## 升级

``` sh
cd ~/.xiaohei/contrib
git pull
pip install --upgrade -r requirements.txt
```
